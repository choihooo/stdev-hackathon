### 미구현 중 영향도 높은 항목 TOP 10

1. **적 방어력/근력 실제 적용** — 적 버프가 데미지에 반영 안 됨 (게임플레이 핵심)
2. **카드 제거 서비스** — 상점에서 덱 정제 불가 (전략성 저하)
3. **이벤트 다양화** — 항상 같은 이벤트만 발생 (재미 저하)
4. **창업자 패시브 실제 효과** — 설명만 있고 작동 안 함
5. **화면 전환 애니메이션** — 모든 전환이 즉시 전환 (몰입감 저하)
6. **수치 팝업/이펙트** — 데미지/힐 수치 변화가 눈에 안 보임
7. **난이도 영향** — 선택해도 게임에 차이 없음
8. **키보드 단축키** — 마우스만 조작 가능
9. **덱 뷰어** — 현재 덱 확인 불가
10. **휴식 업그레이드** — 카드 강화 선택 없이 Focus +1만
